Hide comments with publish dates in the future.

== Description ==

Want to hide comments that are planned in the future?
With this plugin those comments won't be displayed anymore on the front-end.

== Installation ==

1. Upload `wp-hide-future-comments` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How does it work?  =

There is an extra argument given to only show comments with a date before the current datetime.
